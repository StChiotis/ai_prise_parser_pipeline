## Project status

This repository is published for educational use and open-source reference. 

It is not actively maintained. Issues and PRs may receive limited or no response. The code is provided “as-is.”

## Scope

In scope:

- Code in this repository
- Example configuration and schema files

Out of scope:

- Third-party services and models (OpenAI/Azure OpenAI, SQL Server)
- Retailer content, websites, and any source data you provide
- Your deployments, infrastructure, or forks of this repository

## Supported versions

Security fixes are not guaranteed. 

If fixes are made, they target only the latest release.

- Active support: latest release tag (best effort, not guaranteed)
- End of life: all prior releases and commits
